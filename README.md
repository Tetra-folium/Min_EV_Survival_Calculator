# Min_EV_Survival_Calculator
This tool is designed to find the minimum amount of EVs required for a Pokemon to live a given attack. Notably, "Mixed Bulk" can be considered, as the tool can find spreads that live a given physical and special attack with one EV spread.

e.g. Tyranitar with 208 HP / 24 Def / 8 SpD survives a Meteor Mash from 252+ Atk Metagross or a Hydro Pump from 252+ SpA Suicune.

Items and field effects such as Leftovers, Sandstorm, and Spikes have been implemented to simulate turns with higher accuracy. Multiple-attack sequences can be checked as well.

e.g. Blissey to take a Brick Break from Salamence then two Thunderbolts from Zapdos while Sandstorm and Leftovers are active.

Currently, the tool is only accurate to Generation 3 mechanics. Scaling to other Generations may be possible in the future.
